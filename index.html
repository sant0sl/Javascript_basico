Var, Let, Const e String diferente

var (cria variável global)
let (variável que fica restrita onde foi criada)
const (constante: valor fixo, não se altera)
Concatenação diferente com crase para String ---> `Meu nome e: ${variavelNome}`

Desconstrução de Objeto e de Array

Exemplo de um objeto constante de Pessoa
const pessoa = {
nome: "Leonardo",
sobrenome: "Santos",
idade: 23
};

Para pegar somente algumas das informações do objeto pessoa:
let { sobrenome, idade } = pessoa;

Para pegar um valor dentro de um Array e colocar em variável:
let { posição: nomeVariável } = array;

Para pegar mais de um valor dentro de um Array e colocar em variável:
let { posição: nomeVariável, posição: nomeVariável } = array;

Posições em Array para variáveis:
let [ nomeVariável, nomeVariável ] = array;
*Em sequencia 0,1,2..

Map em Array

const lista = [1,2,3,4,5,6];
--- percorre a lista ---
const novaLista = lista.map(function(item){
return item * 5;
});
console.log(novaLista);
--- percorre a lista e somando com o index da vez ---
const novaLista = lista.map(function(item, index){
return item + index;
});
console.log(novaLista);
--- soma todos os itens da lista ---
const soma = lista.reduce(function(total, proximo){
return total + proximo;
});

includes, endsWith e startsWith

includes: função booleana que verifica em uma lista a partir de uma string, a existência ou não de algo dentro da lista
endsWith: função booleana que verifica em uma variável string, se termina com tal parâmetro
startsWith: função booleana que verifica em uma variável string, se começa com tal parâmetro

some e every

Exemplo:

nomes.some(nomes => nomes === 'Leonardo');
